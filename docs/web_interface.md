# Web 可视化工作台

项目提供零额外前端依赖的本地 Web 界面，前端页面和 JSON API 均由 Python
标准库服务提供。页面直接调用现有能力工厂代码，不使用模拟结果。

## 启动

在项目根目录执行：

```powershell
uv run python -m inventory_agent web --open-browser
```

默认地址：

```text
http://127.0.0.1:8000
```

如果浏览器没有自动打开，可手动复制该地址。停止服务时回到运行命令的终端，按
`Ctrl+C`。

端口被占用时可以更换端口：

```powershell
uv run python -m inventory_agent web --port 8080 --open-browser
```

## 页面功能

- **系统总览**：算法、指标、验证配置、知识图谱和历史运行统计；
- **完整能力工厂**：从自然语言需求执行能力检索、候选比较、代码生成、验证修复、
  报告生成和图谱回写；
- **能力抽取与复刻**：从文档、JSON 或 Python 抽取能力，并生成独立可运行代码；
- **算法回测比较**：选择商品、仓库、候选模型和验证配置执行滚动回测；
- **运行证据**：查看候选指标、代码验证矩阵、修复次数和 Agent 工具调用时间线；
- **知识图谱**：嵌入交互式能力图谱；
- **插件与配置**：查看当前注册的算法、指标和任务验证策略。

## 常见问题

### 页面无法访问

确认终端仍在运行 Web 服务，并检查是否显示：

```text
库存算法能力工厂 Web 界面：http://127.0.0.1:8000
```

### 修改前端后没有变化

按 `Ctrl+F5` 强制刷新。首页是单文件自包含页面，服务运行期间修改文件后直接刷新
即可，不需要重新编译前端。

### 端口被占用

使用其他端口：

```powershell
uv run python -m inventory_agent web --port 8081
```

### 是否允许局域网访问

默认只绑定本机。若确实需要在可信局域网中演示，可使用：

```powershell
uv run python -m inventory_agent web --host 0.0.0.0 --port 8000
```

此服务未提供用户登录和权限系统，不应直接暴露到公网。
